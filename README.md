# Portfolio (REACT)

Essa aplicação foi desenvolvida durante o Curso de React do professor Edson Maia do Youtube utilizando React, JavaScript e Sass.

# Como rodar este projeto?
      git clone portfolio

 ## Criação do projeto
    npm create vite@latest

 ## Acesse a pasta do seu projeto no seu terminal
    cd portfolio

  ## Execute a aplicação
     npm run dev  

## A Aplicação será iniciada na porta  5173, acesse pelo navegador:
    http://localhost:5173/      

## Documentações das Ferramentas:
- [vite](https://vitejs.dev/)
- [React](https://react.dev/)
- [Sas](https://sass-lang.com)

# Dependencias Utilizadas:
     npm install Sass
     npm install react-router-dom
